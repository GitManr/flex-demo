子项属性：flex-grow、flex-shrink、flex-basis、flex、align-self、order

flex-grow:

用来按比列分配剩余空间(扩展)，默认值=0
第一个 第二个 第三个 第四个 第五个
0 0 0 0 0

flex-shrink:

当空间不足时，按比例收缩，默认值=1
第一个 第二个 第三个 第四个 第五个
1 1 1 1 1

order:

定义项目的排列顺序，值越小越靠前，默认值为0
第一个 第二个 第三个 第四个 第五个
0 0 0 0 0

flex-basis:

子项在主轴上的基准尺寸，值为百分比或像素值，默认值=auto

注意：配合flex-grow和flex-shrink使用才能发挥效果

flex-basis:100px; 可以

flex:

复合属性：flex-grow、flex-shrink、flex-basis的简写

多个值以空格隔开，按既定的顺序(grow、shrink、basis)，可省略

值得写法有多种形式

flex:none; 不扩展、不收缩的需求。 等同于：flex: 0 0 auto;

flex:auto; 可扩展、可收缩的需求。 等同于：flex: 1 1 auto;

flex:1 0; 只设置的前两个值，basis的值为0%;
